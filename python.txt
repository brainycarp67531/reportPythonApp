====
create a new environment

python3 -m venv env

The last env is the name of the environment, could be anything. 

====
Active the environment

source env/bin/activate   # if not already activated

====
Install packages

pip install PyMuPDF

====
Deactivate the environment

deactivate

====
export list with packages

pip freeze > requirements.txt

====
Install package from requirements file

pip install -r requirements.txt
